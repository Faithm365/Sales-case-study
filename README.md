# Sales Case Study Project
Analysis of the daily trading information for a large retail store, aggregated for the day for only one product in the store.
# Purpose of the case study
- To develop metrics and derive subsequent insights
- To derive any other interesting insights from the dataset provided
- To prepare a report of the findings
# Objectives
- Determine the daily sales price per unit
- Determine the average unit sales price of the product
- Determine the daily %gross profit per unit
- Determine the price elasticity of demand for this product
# Tools and Technologies used
- MIRO for data planning
- Databricks to analyse the data using sql querries
- tMicrosoft excel to create graphs and Tables
- Powerpoint presentation
- Github to load the final presentation
